가계부를 작성하는 프로그램을 만들었다. 날자별로 일정과 사용금액을 등록할 수 있고 사용자의 소비패턴을 그래프로 나타내었다.

오픈소스 프로젝트

가난한 대학생(본인)을 위한 콜벤비 1/n 하는 애플리케이션을 만들었다. 네이버 지도를 이용하여 어디부터 어디까지 갈 것인지 선택하여 방을 개설하고 같은 출발지, 목적지에 해당하는 사람들이 같이 갈 수 있게 하는 애플리케이션이다.

모바일 소프트웨어 프로젝트

사용자가 원하는 인스타그램의 계정 속 인물만 추려내어 저장할 수 있는 애플리케이션을 만들었다. 해당 사용자의 계정에 들어가서 이미지를 크롤링하고 크롤링 된 이미지를 구글 cloud api object decetion 에 전송하여 인물사진만 추려내어 사용자가 확인할 수 있게 하였다. 왜인진 모르겠으나 학교 내에 경진대회도 나갔다.
